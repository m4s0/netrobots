This is the API

robotActionPost

send an action to a robot. The server will answer after the simulation turn is completed, and it will return the token for the next simulation turn.

URL

http://localhost/robot-action

HTTP Method

POST

Response Type

RobotStatus

Parameters

  • Body: command RobotCommand(RobotCommand)

robotCreatePost

Create a new robot.

URL

http://localhost/robot-create

HTTP Method

POST

Response Type

RobotStatus

Parameters

  • Body: configuration RobotConfiguration(RobotConfiguration)

robotRemovePost

Remove a robot.

URL

http://localhost/robot-remove

HTTP Method

POST

Response Type

Parameters

  • Body: remove RemoveCommand(RemoveCommand)
